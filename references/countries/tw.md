## ANALYST 가 대만 조사 시 권장 순서

1. **거시·산업 규모** → DGBAS / MOEA / 中華經濟研究院
2. **상장 플레이어 재무** → MOPS 年報 + 月營收 (월간 매출은 대만만의 강점)
3. **ICT·SW 시장** → 資策會 MIC (가장 인용 많음) + Mordor / Statista cross-reference
4. **반도체·하드웨어** → TrendForce + 工研院 IEK
5. **스타트업·비상장** → 數位時代 (Bnext) + INSIDE + 商業司 사업자등록
6. **국제 비교** → World Bank / IMF (단, 대만은 일부 국제기구 데이터에 빠짐 — 별도 확인)

## 대만 시장 조사 시 흔한 함정

- ❌ 통화: 대만달러 (TWD, NT$). USD 환산 시 시점 명시
- ❌ 표기: 中華民國 / 台灣 / Republic of China — 자료마다 표기 다름. 국제기구 자료는 "Chinese Taipei" 일 수 있음
- ❌ 일부 국제기구 (UN, WTO 일부) 데이터에서 대만이 빠짐 → 자체 통계 (DGBAS) 가 더 정확한 경우 많음
- ❌ 民國紀年 (Minguo calendar) 사용. 西元 (서기) 환산 필요. 民國 113년 = 2024년
- ❌ 회계연도가 캘린더와 일치 (1-12월) — 일본과 다름
- ❌ "中文版 vs 英文版" 데이터 차이 — 中文版이 더 최신·정확한 경우 많음

## 대만 특유의 데이터 강점

- **반도체·하드웨어**: 글로벌 최강. TrendForce / 工研院 IEK / DIGITIMES 가 1차 출처
- **월간 매출 공시**: 매월 10일까지 — 트렌드 분석에 유용
- **電子發票 (전자영수증)**: 2014년부터 의무화 → 일부 매출 데이터 추적 가능

전자·반도체·디스플레이 영역에서 대만은 글로벌 데이터의 1차 생산지. 다른 국가 데이터를 그쪽에서 역으로 확인하는 경우도 많습니다.
